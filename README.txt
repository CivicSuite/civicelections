CivicElections v0.1.1 ships election administration support foundations plus optional database-backed candidate/canvass workpapers through CIVICELECTIONS_WORKPAPER_DB_URL. It is not a voter registration system, ballot marking device, tabulator, election conduct automation system, campaign finance system of record, official certification workflow, live LLM runtime, or election-system connector. Dependency: civiccore==0.3.0.
